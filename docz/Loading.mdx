import Loading from 'components/Loading';

# Loading

## Signature

```
<Loading failed={Boolean} data={String} />
```

## Examples

## While loading state

```
<Loading />
```

<Loading />

## Failed to load state

```
<Loading failed={true} data={'Error while loading'} />
```

<Loading failed={true} data={'Error while loading'} />

## Sucessfully loaded

`data` is ignored when the component sucessfully loads

```
<Loading failed={false} data={'Loaded!'} />
```

<Loading failed={false} data={'Loaded!'} />
